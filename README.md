# ブロック崩しゲーム

このプロジェクトは、HTML5 Canvasを使用して作成されたクラシックなブロック崩しゲームです。モダンなデザインと滑らかなアニメーションを特徴としています。

## 機能

- 美しいグラデーションを用いたモダンなUIデザイン
- スムーズな操作性（キーボードとマウス対応）
- スコアと残機のトラッキング
- 5行×9列のカラフルなブロック配置
- ゲームの進行に応じたボールの速度調整
- ゲームオーバーとクリア時の演出

## 操作方法

- **パドルの移動**: 
  - 左右矢印キー
  - または、マウスでパドルを直接操作
- **ゲームの開始/一時停止**: スペースキー
- **ゲームのリスタート**: 「もう一度プレイ」ボタン（ゲーム終了時）

## 技術仕様

- **フロントエンド**:
  - HTML5
  - CSS3（アニメーションとグラデーション効果）
  - JavaScript（Canvas API）

## ファイル構成

- `index.html` - メインのHTML構造
- `style.css` - ゲームのスタイリング
- `game.js` - ゲームのロジックとアニメーション

## ゲームの特徴

1. **段階的な難易度**:
   - ボールがパドルに当たるたびに徐々に速度が上昇
   - 最大速度の制限あり

2. **視覚的フィードバック**:
   - パドルとボールの発光効果
   - スムーズなアニメーション
   - ブロック破壊時の即時反応

3. **スコアシステム**:
   - 各ブロックの破壊で10ポイント獲得
   - 3回のチャンス（残機）
   - 全ブロックの破壊でゲームクリア
